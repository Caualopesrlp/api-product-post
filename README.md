# 🚀 Product API POST

Projeto de estudo desenvolvido em Laravel para praticar integração com APIs externas utilizando requisições HTTP do tipo POST.

O objetivo foi entender o fluxo de envio de dados para uma API, tratamento de respostas, validações e tratamento de erros.

---

## 🎯 Objetivo do projeto

Praticar:

- Consumo de APIs REST
- Envio de dados via POST
- Payload JSON
- Laravel HTTP Client
- Tratamento de erros
- Logs
- Form Requests
- Service Layer
- Feedback para o usuário

---

## 🛠️ Tecnologias utilizadas

- PHP 8+
- Laravel 12
- Blade
- CSS
- Laravel HTTP Client

---

## ✨ Funcionalidades

- Formulário para criação de produtos
- Validação dos dados enviados
- Envio dos dados para uma API externa
- Tratamento de erros da API
- Logs de falhas
- Mensagens de sucesso e erro para o usuário

---

## 🌐 API utilizada

Foi utilizada a API de testes DummyJSON:

https://dummyjson.com

Endpoint utilizado:

POST https://dummyjson.com/products/add
